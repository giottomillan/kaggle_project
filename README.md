## 🔍INFO DE COLUMNAS 

*   `Manufacturer` - Brand
*   `Model Name` - Model Name / Number
*   `Category` - Type of laptop
*   `Screen Size` - Screen Size in inches
*   `Screen` - Screen Type
*   `CPU` - CPU type
*   `RAM` - RAM quantity
*   `Storage` - Storage capacity
*   `GPU` - GPU type
*   `Operating System` - OS
*   `Operating System Version` OS Version
*   `Weight` - Weight (kg)
*   `Price` - Price (currency: Sonic Rings) 😜
